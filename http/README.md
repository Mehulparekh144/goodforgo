# HTTP Project

A simple HTTP server project written in Go.

## Prerequisites

- Go 1.24.3 or later

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/Mehulparekh144/goodforgo.git
cd http
```

2. Build the project:

```bash
go build ./cmd/...
```

3. Run the server:

```bash
./http
```

## Project Structure

```
.
├── cmd/        # Command-line applications
├── go.mod      # Go module file
└── README.md   # This file
```
